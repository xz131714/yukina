# RyuChan中的MDX使用指南

RyuChan是一个基于Astro框架的现代博客系统，它充分利用了MDX的强大功能，允许您在Markdown文档中嵌入交互式组件。本指南将向您展示如何在RyuChan中充分利用MDX的各种特性和组件。

## 什么是MDX?

MDX是Markdown的超集，它允许您在Markdown内容中直接使用JSX语法和组件。这意味着您可以在编写文档的同时，插入交互式UI组件，使您的内容更加生动和有吸引力。

<Info>MDX = Markdown + JSX，让您的内容具有更强的表现力和交互性！</Info>
